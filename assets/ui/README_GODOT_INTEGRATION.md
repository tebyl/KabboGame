# Integración rápida en Kabbo

Copiar carpeta completa a:

res://assets/ui/

Ejemplo con TextureButton:
- normal: buttons/btn_blue_normal.png
- hover: buttons/btn_blue_hover.png
- pressed: buttons/btn_blue_pressed.png
- disabled: buttons/btn_blue_disabled.png

Ejemplo con NinePatchRect:
- Texture: panels/panel_blue_9slice.png
- Patch Margin: 8 px

Íconos recomendados:
- icon_home.png para top bar / Mis salas
- icon_chat.png para panel chat
- icon_coin.png para monedas
- icon_inventory.png para inventario
- icon_decorate.png para decorar
- icon_close.png para cerrar
- icon_search.png para búsqueda
