# Demo Guide

## Objetivo

Esta demo local presenta el loop completo de KabboLike: entrar a una sala isometrica, moverse, decorar, comprar muebles, administrar salas, conversar localmente y comprobar persistencia.

## Recorrido sugerido

1. Abrir la demo sin save para ver onboarding.
2. Caminar por la sala con click.
3. Abrir Inventario y activar Decora.
4. Colocar, mover, rotar y eliminar un mueble.
5. Comprar un mueble en Tienda y colocarlo.
6. Abrir Menu > Editar sala y cambiar piso/pared.
7. Abrir Menu > Salas, crear otra sala y volver.
8. Abrir Menu > Info sala y editar perfil de sala.
9. Enviar chat local y observar NPCs.
10. Abrir Misiones, reclamar recompensa y revisar Logros.
11. Guardar, cerrar y reabrir para confirmar persistencia.

## Controles utiles

- Click en piso: caminar.
- Enter: enfocar chat.
- Inventario: iniciar preview de muebles.
- Decora: activar edicion de sala.
- Menu: info, editar sala, guardar, audio, tutorial, reset y salida.

## Bugs conocidos / limites de demo

- No hay multijugador, login ni visitas reales online.
- NPCs son simulados localmente.
- Chat es local y temporal; no se guarda entre sesiones.
- Economia y progreso viven en save local.
- Export/import usa archivos locales.
- Duplicar sala no descuenta inventario.
- Avatar no es todavia un sistema de capas completo.

## Features no incluidas

- Servidor o matchmaking.
- Marketplace.
- Comentarios online.
- Ranking global.
- Guardado cloud.
